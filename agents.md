# Project Agents: Global-BioScan

## 1. Bio-Data Engineer (Standardization & Cleanup)
- **Responsibility:** Standardizing raw FASTA/FASTQ files.
- **Tools:** TaxonKit, Biopython, Fastp.
- **Focus:** Resolving taxonomic clashes and formatting data for TPU ingestion.

## 2. ML Scientist (TPU & Foundation Model)
- **Responsibility:** Fine-tuning and embedding generation.
- **Tools:** JAX, Flax, Hugging Face PEFT (LoRA).
- **Focus:** NT-2.5B optimization, Hierarchical Loss functions, and TPU-to-GCS streaming.

## 3. Database Architect (Vector Search & Scale)
- **Responsibility:** Managing the 2TB SSD storage.
- **Tools:** LanceDB, PyArrow, FAISS.
- **Focus:** Disk-based indexing, IVF-PQ optimization, and metadata filtering.

## 4. Bioinformatician (Ecological Analytics)
- **Responsibility:** Novelty discovery and diversity metrics.
- **Tools:** HDBSCAN, Scikit-learn, MAFFT (Phylogeny).
- **Focus:** Alpha/Beta diversity, cluster centroid analysis, and novelty thresholding.

## 5. UI/UX Developer (Dashboard)
- **Responsibility:** Scientific visualization.
- **Tools:** Streamlit, Plotly, Leaflet/Folium.
- **Focus:** Latent space mapping (UMAP) and interactive sunburst charts.

## 6. Demo Specialist (Funding & Visualization)
- **Responsibility:** Creating the "Simulated Scale" environment for stakeholders.
- **Focus:** User experience, high-level metrics, and "The Discovery Moment" workflow.
